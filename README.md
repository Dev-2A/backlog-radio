# 📻 Backlog Radio

> 안 한 게임의 OST부터 들어보기 — Steam 백로그 게임 사운드트랙 자동 발굴

Steam 라이브러리에서 **구매했지만 거의 안 한** 게임들의 사운드트랙을 자동으로 찾아 작업 BGM 플레이리스트로 만들어주는 도구.

"이 음악 좋네, 게임도 한번 해볼까?" — 백로그를 강제로 플레이하라고 다그치는 대신, 음악으로 부드럽게 입문하게 만드는 게 목표야.

## ✨ 주요 기능 (예정)

- 🎮 Steam ID 입력만으로 라이브러리 자동 분석
- 📦 플레이타임 기준 백로그 자동 필터링
- 🎵 게임별 OST YouTube 자동 검색
- 🎯 매주 추천 — "이번 주의 백로그"
- 🎧 미니 플레이어 + 작업 모드 (작업하면서 듣기)
- 📊 들은 OST 히스토리 + 백로그 진행도 통계

## 🔧 기술 스택

- **Next.js 15** (App Router) + JavaScript
- **Tailwind CSS v4**
- **Steam Web API** — 라이브러리/플레이타임 조회
- **YouTube Data API v3** — OST 검색
- **Vercel** — 배포 + 서버리스 함수
